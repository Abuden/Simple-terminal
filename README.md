# Simple terminal

Implemented a simple version of the terminal using C.

## Features

```
ls
```

```
sleep 1
```

```
pwd
```

```
mkdir test_dir
```

```
cd test_dir
```

```
pwd
```

```
cd ../
```

```
rmdir test_dir
```

```
echo "hello world"
```

```
echo "test text" > file1
```

```
whoami
```

```
cat file1
```
